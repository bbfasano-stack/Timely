# Timely

A smart task prioritization feed web app.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Open your browser to `http://localhost:5173` to view the app.

## Features

- Smart Task Feed: View one task at a time, swipe or scroll vertically to navigate.
- Actions: Start Now, Schedule It, Skip.
- Mobile-first design with dark mode.
